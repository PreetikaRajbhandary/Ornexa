# Orenxa – Jewelry E-Commerce Web Application

## About Project

Orenxa is a group-based full-stack jewelry e-commerce web application developed to provide users with an online shopping platform and administrators with inventory and product management capabilities.

## Technologies Used

* Java
* JSP
* Servlets
* MySQL
* HTML
* CSS
* JavaScript
* MVC Architecture

## Features

* Product browsing and search
* Order placement functionality
* Admin management panel
* Product and inventory management
* Session-based authentication
* Database connectivity

## Process (How We Built It)

The project was developed collaboratively by designing the application architecture and separating responsibilities across frontend, backend, and database development. JSP and Servlets handled application functionality while MySQL managed data storage following the MVC architecture.
